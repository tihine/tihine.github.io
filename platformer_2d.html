Platformer 2D

School project

This project is being developed as part of my studies. There are 3 of us working on it. The aim is to create a platformer that can be controlled with xbox controllers and that is interesting for the player.

I'm mainly interested in horizontal mechanics: movement, sprinting and dash. I also created the visual effects linked to these mechanics. These mechanics use an 'energy' resource, visible in real time to the player via an in-game UI bar.
